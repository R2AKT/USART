# USART
Serial port module (USART). For connection to the CPU_8080 processor board - https://github.com/R2AKT/CPU_8080.

Based on 580VV51 (8251) and 580VI53 (8253).

Status: Tested. Prescaler x1 and x16, speed up to 38400.
